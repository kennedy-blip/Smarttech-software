# 📱 SmartTech Pro Repair Suite (2026 Edition)

SmartTech Pro is a professional-grade Android utility designed for mobile repair technicians. It streamlines hardware diagnostics, security scanning, data protection, and advanced recovery operations into a single, portable interface.

## ✨ Key Features
* **🔒 FRP Bypass:** Advanced scripts to bypass Google Factory Reset Protection locks.
* **💾 Safe Backup:** One-click media and document extraction before device wipes.
* **🧹 System Optimizer:** Bulk bloatware removal and cache cleaning to restore speed.
* **🔋 Diagnostics:** Real-time battery health and hardware status monitoring.
* **📄 Professional Invoicing:** Automated PDF report generation for customer billing.
* **🚀 Portable:** Compiled into a single `.exe` for use on any Windows technician workstation.

## 🛠️ Tech Stack
* **Language:** Python 3.14+
* **UI Framework:** PyQt6 (Modern Dark Theme)
* **Engine:** Pure-Python-ADB (PPADB)
* **Report Engine:** ReportLab
* **Bundler:** PyInstaller

## 🚀 Getting Started (for Developers)
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/smarttech.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python main.py`

## 📲 Connection Guide
To use this tool, the Android device must have:
1. **Developer Options** enabled (Tap 'Build Number' 7 times).
2. **USB Debugging** toggled ON.
3. **RSA Key Trusted** (Select 'Always allow from this computer' on the phone popup).
